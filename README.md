# batch-sound-round

## 说明
环境音播放工具
* 下载[环境音](https://pan.baidu.com/s/1sl6UKsl)，存储本地硬盘
* 修改soundRound.bat中的soundBase变量为环境音文件夹
* 执行soundRound.bat
* 使用上下键移动光标，使用回车键控制播放与否，可以同时播放多个环境音
* 更加具体的说明可以查看这个[地址](http://www.nexttoyou.me/index.php/archives/163)
